3. **Templates Dashboard** - Application-wise template catalog and library

### Knowledge Base Dashboard

### BA Review Dashboard
- Prompt review queue from KB Chat
- Approve/reject workflow

- Metadata tracking
- Export functionality

### Templates Dashboard
- Application-wise template catalog
- Template details and preview
- Template download as JSON
- Template management (archive, delete)
- Search and filter
- Understanding template purpose through description and content
