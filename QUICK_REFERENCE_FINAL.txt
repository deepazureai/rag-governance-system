IsImprovementSaved Flag (0 or 1):
  ├─ Stored in RawDataRecord.baReview
  ├─ Loaded on modal open via GET /recommendations
  ├─ Set to 1 when improvements saved
  ├─ If 1: Generate button disabled, Add Improvement hidden
  └─ Persists across close/reopen cycles

Data Storage:
  ├─ All data in single RawDataRecord collection
  ├─ No separate recommendations collection needed
  ├─ Query by applicationId + recordId
  └─ Flag + all fields returned in response


BUTTON BEHAVIOR

  Flag = 0 → ENABLED (click to generate)
  Flag = 1 → DISABLED, shows "Improvements Saved ✓"

Add Improvement:
  Flag = 0 → VISIBLE (click to edit)
  Flag = 1 → HIDDEN, shows "Cannot modify further"


WORKFLOW EXAMPLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Open Modal (First Time):
  → GET returns: flag=0
  → Generate button: ENABLED
  → Add Improvement button: VISIBLE
  → Click Generate → Recommendations show
  → Click Add Improvement → Edit form
  → Click Save → flag set to 1

Close Modal

Reopen Modal (Same Record):
  → GET returns: flag=1 + all saved data
  → All previous data displays
  → Generate button: DISABLED
  → Add Improvement button: HIDDEN
  → User cannot modify
